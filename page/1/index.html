Retrouvez votre bien-être en quelques jours.

REVITABIOME est un programme d’exploration alimentaire basé sur les dernières connaissances scientifiques sur le microbiome intestinal, pour retrouver votre bien-être rapidement.

Il est conçu pour limiter les troubles des personnes souffrant de:

  • Syndrome de l’Intestin Irritable (SII)
  • déséquilibre du microbiome comme le SIBO
  • Rectocolite Hémorragique
  • Candidose Chronique
  • Intolérances Alimentaires
  • Fatigue Chronique
  • Douleurs Articulaires et Fibromyalgie
